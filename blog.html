Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM is 'getElementById','getElementsByClassName','getElementsByTagName','getElementsByName','querySelector','querySelectorAll'.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML, innerText, and textContent are used to get or set the content of an element, but they have important differences:
1.'innerHTML' gets or sets HTML markup,
2.'innerText' gets or sets visible text (affected by CSS),
3.'textContent' gets or sets all text (ignores styling, includes hidden text).

Question-3: What is event delegation in the DOM?

Event delegation means adding one event listener to a parent element to handle events from its child elements, instead of adding listeners to each child.

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event happens on a child element, it automatically moves up to its parent elements one by one.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, We can create a new HTML element using document.createElement("tagName"), then set its content or attributes (like textContent, className, or id). To add it to the page, we use methods like appendChild(), append(), or prepend() to insert it into a parent element.
Once added, if we want to remove it later,we can use element.remove() or parent.removeChild(element) . This allows to dynamically build, modify, or clean up parts of a webpage using JavaScript.
